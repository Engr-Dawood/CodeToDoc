# Lightweight LLM for Code-to-Document Generation

A lightweight, locally deployable Large Language Model (LLM) that assists developers in generating high-quality technical documentation from source code.

## Features

- Automatically generate documentation from source code
- Interactive prompting for missing or ambiguous information
- Customizable document format templates
- Optimized for speed, accuracy, and resource efficiency
- Local deployment without requiring cloud services

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/code-to-doc-llm.git
cd code-to-doc-llm

# Install dependencies
pip install -r requirements.txt

# Install the package
pip install -e .
